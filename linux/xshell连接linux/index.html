2.安装linux ssh client

方法如下:

  • 1.查看linux系统是否开启ssh服务，终端输入ps -e | grep ssh 如果服务已经启动，则可以看到 “sshd”，否则表示没有安装服务，或没有开机启动。
  • 2.若没有安装则安装ssh服务，输入命令：sudo apt-get install openssh-server openssh-client
  • 3.启动服务: /etc/init.d/ssh start

3.连接SSH

方法如下:

  • 1.打开xshell 并新建一个 会话。
  • 2.在会话窗口 在主机对应的编辑框输入 之前记录的IP。
  • 3.之后输入自己的用户名已经密码就可以搞定。
